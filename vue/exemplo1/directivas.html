Notas rápidas:

  • v-bind: enlaza un atributo HTML con un valor de Vue (solo Vue → HTML).
  • v-model: enlaza un input o select de manera bidireccional (Vue ↔ HTML).
  • Con v-bind:[atributo] puedes elegir dinámicamente qué atributo actualizar.
  • Con v-model cualquier cambio en el input o select se refleja automáticamente en Vue.
